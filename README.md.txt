Projeto: Análise de Dose em Exames Radiológicos
Objetivo

Este projeto tem como objetivo simular um banco de dados hospitalar para analisar a dose de radiação acumulada por pacientes submetidos a exames radiológicos.

A proposta é aplicar conceitos de SQL e Python para:

Modelagem de banco de dados

Consultas analíticas

Cálculo de dose acumulada

Identificação de padrões e possíveis riscos

Contexto

O controle de dose em exames radiológicos é um tema relevante em Física Médica, pois envolve segurança do paciente e monitoramento de exposição à radiação ionizante.

Este projeto simula um cenário hospitalar fictício para fins educacionais.
